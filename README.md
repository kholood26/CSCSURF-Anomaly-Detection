# CSCS-URF: Anomaly Detection for Healthcare IoT

This repository provides the implementation of the Customized Sand Cat Swarm driven Updated Random Forest (CSCS-URF)
framework for anomaly detection in healthcare IoT ecosystems.

## Features
- Min-Max normalization
- Recursive Feature Elimination (RFE)
- Customized Sand Cat Swarm Optimization (CSCSO)
- Updated Random Forest (URF)
